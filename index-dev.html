Instacam is a library to perform instant canvas video through the WebRTC API with a fresh touch of CSS filters. With Instacam, you can do much more than display a camera stream! CSS and custom filtering applied on HTML5 canvas, snapshots, etc. View the project on Github to see the documentation about available options, methods and callbacks.

CSS filter

Adds pretty css filters to the camera

Custom filter

Creates custom filters based on the canvas magic

Snap

Takes a snapshot of the camera

Save

Exports the camera stream in different format and quality
