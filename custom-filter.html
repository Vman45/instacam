Instacam

Instacam is a jQuery plugin to perform instant canvas video through the WebRTC API with a fresh touch of CSS filters.

Documentation

Instacam can do much more than display a camera stream! CSS filtering, custom filtering applied on HTML5 canvas, snapshots, etc. View the project on Github to see the documentation about available options, methods and callbacks.
